# Craftygiftsplace Site

Estrutura base do site da loja Craftygiftsplace.

## Pastas principais

- `assets/css`: estilos
- `assets/js`: scripts
- `assets/img`: imagens, icones e logotipo
- `assets/docs`: notas internas da marca
- `pages`: paginas internas do site
- `data`: dados de produtos em JSON

## Nota

Os ficheiros de imagem e favicon foram criados como placeholders para substituir depois pelos assets finais.
